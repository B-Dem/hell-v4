 <h1> Manual Deploy</h1>
 
<h5> Clone the repo using git :</h5>

```
$ git clone https://github.com/B-Dem/hell-v4
```

<h5> Install with :</h5>

```
$ npm install
```

<h5> Start the server with :</h5>

```
$ npm start
```
